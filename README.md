# Fortis Pricing System (FPS)

Sistema interno de precificação desenvolvido para a Fortis.

## Objetivo

Centralizar o processo de:

- Precificação
- Engenharia de produtos
- Aprovação comercial
- Histórico de propostas
- Relatórios

## Tecnologias

Frontend
- HTML5
- CSS3
- JavaScript ES6+

Backend (em desenvolvimento)
- Node.js
- Express

Banco (planejado)
- SQLite
- PostgreSQL
